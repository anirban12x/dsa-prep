# DSA Master Tracker

DSA Master Tracker is a high-fidelity, offline-first preparation tracker combining three top coding sheets into one unified, distraction-free environment:

* **TCS NQT Coding Sheet**
* **Blind 75**
* **NeetCode 150**

## Features

* 🚀 **Default Dark Mode**: Permanent, programmer-centric aesthetic.
* 🎨 **Multi-Palette Accents**: Switch accent color themes dynamically between Cyan, Purple, Emerald, and Amber.
* 📚 **Beginner-Friendly Pointwise Solutions**: Detailed, step-by-step optimal recipes and mathematical complexity analysis proofs for every algorithm.
* 💻 **VS Code Simulated Java IDE**: High-fidelity tabbed interface featuring syntax highlighting, compiles badge, copy shortcuts, and line numbering.
* 🛡️ **Offline-First Persistence**: Save your progress, bookmarks, and search state instantly using Local Storage.
* 📥 **Import & Export**: Backup and restore your coding progress at any time.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```

3. **Production Build**:
   ```bash
   npm run build
   ```

## Stack

* **Frontend**: Next.js / React
* **Router**: TanStack Router & React Query
* **Styling**: Vanilla Tailwind CSS & custom CSS variables
* **Icons**: Lucide React
